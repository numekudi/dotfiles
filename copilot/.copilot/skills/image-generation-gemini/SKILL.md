---
name: Image Generation Skill
description: Generates an image based on the user's input.
---

## usage
```sh
uv run nskill image-generation <prompt> <output_path> [options]
```
### required arguments

- `prompt`
- `output_path`

### optional arguments

- `--reference-image <path>`
- `--model <model_name>`
- `--temperature <float>`
- `--image-size <size>`
- `--aspect-ratio <ratio>`
